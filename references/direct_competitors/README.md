## 3. 빈 좌표 (NoF 기회)

폴리시드한 quit-porn 직접 경쟁사 비주얼은 사실상 QUITTR 단독. 직접 경쟁 지형은 (a) 공포·수치·남성성 또는 (b) 종교·감시로 양분. **차분·비감시·한국 네이티브·따뜻한 프리미엄** 좌표가 비어 있음 → NoF가 비주얼 리더가 될 공간.
