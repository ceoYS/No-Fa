# CHARACTER_SYSTEM_SPEC.md

캐릭터 성장 시스템 — 리텐션·정서 지지 (처벌·pay-to-win 아님)

상위 문서: PRD §11 / 같은 라운드: UX_RULES.md, MONETIZATION_POLICY.md, COPY_POLICY.md
이 문서는 캐릭터 시스템의 추론·검수 기준이다. 캐릭터 디자인·에셋 생성은 하지 않는다.

---

## 1. 캐릭터 역할 (Character role)

캐릭터는 사용자를 평가하는 존재가 아니라, **자기통제 여정을 시각화하는 동료**다 (PRD §11.1).
목적: 리텐션(매일 열 이유) + 정서적 지지. 처벌·경쟁·과금 도구가 아니다.

---

## 2. 권장 세계관 (Recommended worldbuilding)

PRD §11.3 추천: **Option B + Option C 혼합** (NEEDS DECISION #12, #13 — 사람 확정 전까지 후보).

| 요소 | 방향 |
|---|---|
| 캐릭터 | 절제 환경을 함께 지키는 수호자/동료 |
| 홈 공간 | 사용자의 절제 루틴이 쌓이는 "내 절제 공간" |
| 성장 방식 | 스트릭·체크인·위기 극복·회복 플랜 |
| 꾸미기 | 캐릭터 외형·공간 테마·오브젝트 |

---

## 3. 캐릭터 성격 (Character personality)

- 차분하고 지지적. 동료 같은 거리감.
- 비난·평가·훈계하지 않음. 실패해도 곁에 머묾.
- 위기·실패 순간엔 짧고 따뜻하게. 성취 순간엔 담담히 인정.

---

## 4. 성장 원천 (Growth sources)

PRD §11.4. 행동의 결과로만 성장. 결제로 성장 금지.

| 사용자 행동 | 캐릭터 반응 | 보상 |
|---|---|---|
| 하루 유지 | 안정 상태 유지 | XP |
| 체크인 완료 | 오늘 감정 기억 | XP |
| 위기 버튼 사용 | 함께 버틴 메시지 | XP |
| 대체 활동 완료 | 에너지 회복 | XP |
| 실패 후 회복 플랜 | 다시 시작 응원 | 회복 배지 |
| 7일 스트릭 | 1단계 성장 | 진화/스킨 |
| 30일 스트릭 | 장기 성장 | 테마/칭호 |

---

## 5. 성장 단계 (Growth stages)

- character_stage는 predefined_enum (PRD §18).
- P0: 기본 단계 + XP 누적 표시.
- 진화/단계 전환은 P1 (REQ-012).
- 강등 없음. 실패로 단계가 내려가지 않는다 (PRD §11.5).

---

## 6. 실패 상태 처리 (Failure state handling)

PRD §11.5.

| 금지 | 허용 |
|---|---|
| 캐릭터 죽음 | 캐릭터가 쉬는 상태 |
| 벌점 | 회복 미션 |
| 성장 단계 강등 | 오늘의 회복 기록 |
| 비난 메시지 | 재시작 응원 |
| 공개 실패 알림 | 사용자 선택 기반 비공개 기록 |

---

## 7. 보상 규칙 (Reward rules)

- 보상=행동의 결과. "운/뽑기"가 아니라 행동 누적.
- 위기 버튼 사용도 보상 대상 (사용을 깎지 않음).
- 보상 수령 화면에 "한 번 더 뽑기"·과금 압박 금지 (COPY_POLICY §11).

---

## 8. 꾸미기 규칙 (Customization rules)

- P1: 캐릭터 외형·공간 테마·오브젝트 (REQ-013).
- 꾸미기는 미적 선택이지 능력·성공률에 영향 없음.
- 유료 꾸미기는 P2이며 **비랜덤 단품/테마만** (MONETIZATION_POLICY).

---

## 9. 과금 경계 (Monetization boundaries)

PRD §11.3, §12.3 기준. 금지:
- 캐릭터 사망/실패 처벌
- 전투력·능력치·의지력 점수
- 유료 성장 부스팅
- 유료 스트릭 복구 / 실패 삭제
- pay-to-win 효과 (절제 성공·차단 해제를 돈으로)

단계:
- **P0**: 기본 캐릭터 성장 (행동 기반 XP).
- **P1**: 진화 / 꾸미기.
- **P2**: 유료 코스메틱 — 검증 후, 비랜덤만.

## 10. 카피 예시 (Copy examples)

좋은 예:
- "오늘도 함께 버텼어요."
- "다시 시작 응원할게요."
- "잠시 쉬어가도 괜찮아요."
- "7일을 함께 쌓았어요."

나쁜 예 (금지):
- "캐릭터가 죽었습니다."
- "당신 때문에 실망했어요."
- "뽑으면 절제력이 올라갑니다."

## 12. 사람 검수 체크리스트 (Human review checklist)

- [ ] 캐릭터가 사용자를 비난·평가하지 않는가?
- [ ] 실패 시 죽음/강등/벌점이 없는가?
- [ ] 성장이 행동 기반이며 결제로 살 수 없는가?
- [ ] 전투력·의지력 점수가 없는가?
- [ ] 유료 요소가 비랜덤·비능력(P2)으로 한정되는가?
- [ ] 위기 사용을 보상하고 깎지 않는가?
- [ ] 세계관이 NEEDS DECISION으로 표기되고 임의 확정되지 않았는가?
