## 4. 새 캐릭터 방향: Ember Cat / 잉크 고양이 / 잔불 수호자

추천 방향:

NoF는 **"어두운 잉크 공간 속에서 작은 고양이 수호자가 내 절제 공간을 함께 지켜주는 앱"**으로 잡는다.

캐릭터 이름 / 방향 후보:
- Ember Cat
- 잉크 고양이
- 잔불 수호자
- 조용한 동료

시각 원칙:
- 검은 잉크 실루엣의 작은 고양이
- 눈, 꼬리 끝, 숨결, 주변 잔불에만 앰버(ember) 포인트
- 귀엽지만 유치하지 않게
- OPAL의 gem mascot처럼 앱 전체를 기억나게 하는 상징 구조는 **참고**하되 복제하지 않는다
- 차용하는 것은 "하나의 상징 오브제가 앱 전체의 기억점이 된다"는 구조뿐

## 5. 절대 금지

- 동글동글 아기고양이 펫게임화
- 큰 눈, 볼터치, 과한 표정
- 실패 / 재발 시 고양이가 울거나 죽거나 퇴화
- 전투력, 레벨업, 랭킹 강등
- 확률형 뽑기
- 스트릭 복구권, 실패 삭제권, 차단 해제권
- 캐릭터가 사용자를 감시하거나 판단하는 톤

## 6. 성장 시스템 방향

성장 기준은 현재 스트릭만으로 만들지 않는다.
실패 / 재발이 있어도 캐릭터가 **퇴화하지 않아야 한다.**

성장 입력:
- 체크인 누적
- 위기 버튼 사용 후 버틴 횟수
- 차단 세션 설정 횟수
- 회복 루프 완료 횟수
- 연속일 + 총 기록일

성장 표현:
1. 작은 잉크 고양이 — 눈빛만 작게 켜짐
2. 곁에 작은 잔불이 생김
3. 고양이 주변 공간이 안정됨
4. 꼬리 끝 빛이 길어짐
5. 사용자의 절제 공간 / 방 / 결계가 완성됨

핵심:
- 고양이가 "강해지는 것"이 아니라, **사용자의 절제 공간이 안정되는 것.**
- 캐릭터는 처벌자가 아니라 조용한 동료다.
